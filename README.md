# 🖋️ KAKOKI Creative Co.
### High-Authority Local SEO & Creative Branding Engine

Welcome to the official repository for **KAKOKI Creative Co.**, the premier web design and branding agency based in **Malappuram, Kerala**. This project serves as a high-performance local SEO hub designed to dominate the regional market through a sophisticated search architecture and a premium "sketch-style" creative aesthetic.

---

## 🚀 Key Features & Overhauls

### 🏛️ Local SEO Hub & Spoke Architecture
We have implemented a powerful SEO system that targets the entire Malappuram district:
- **Master Location Hub**: `service-areas-malappuram.html` acts as the primary authority page.
- **Location Spokes**: Dedicated landing pages for **Kondotty**, **Areekode**, and **Malappuram** optimized for local keyword density.
- **Global NAP Footer**: A standardized footer with Name, Address, and Phone data across all pages for consistent local citations.

### 🎯 High-Conversion Lead Capture
- **Sudden Inquiry Popup**: A sleek, bottom-right slide-in form that appears 1 second after page load to maximize lead extraction.
- **Production EmailJS Pipeline**: A unified `contact.js` handler (now in `assets/js/`) that connects both the main contact page and the mini-popup to your professional inbox.
- **WhatsApp Integration**: Direct one-click chat buttons placed strategically across all landing pages.

### 🎨 Premium Creative Aesthetic
- **Infinite Team Loop**: A high-performance, custom JavaScript carousel for team members that bypasses native scroll-snap issues.
- **Sketch-Border UI**: A unique visual language combining thick hand-drawn borders and bold shadows (Tailwind-native).
- **Localized Testimonials**: Replaced generic placeholders with authentic Keralite business success stories and generated local profile portraits.
